== Description ==

Image Map Pro is a plugin that lets you create interactive images for your WordPress website. Load your image and draw shapes on top of it - it's that simple!

Features:

* Custom Polygon Shapes (limited to 1)
* Over 150 SVG Icons Included
* Super Advanced Backend Editor
* Fully Responsive
* Click/Mouseover Actions
* Custom Styles for Shapes
* Image Map Export
* Undo/Redo Functionality
* Zooming in the Editor
* Fullscreen Capability
* Rename & Reorder Shapes
* Copy/Paste & Duplicate Shapes

This is a Lite version, for the full version, please visit [https://codecanyon.net/item/image-map-pro-for-wordpress-interactive-image-map-builder/2826664](https://codecanyon.net/item/image-map-pro-for-wordpress-interactive-image-map-builder/2826664)

Premium version features:

* Unlimited Polygon Shapes
* Connected Shapes
* Content Builder for the Tooltips
* JavaScript & HTML API
* Customer Support for 6 Months
* Import and Export Image Maps

== Installation ==

Install Image Map Pro just like any other WP plugin:

1. Upload the plugin files to the `/wp-content/plugins/image-map-pro-wordpress-lite/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings -> Image Map Pro Lite screen to configure the plugin
